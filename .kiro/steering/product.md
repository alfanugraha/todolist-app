# Product

## Deskripsi

Aplikasi frontend sederhana untuk mengelola daftar tugas (todo list). Berjalan sepenuhnya di sisi klien (browser) tanpa backend atau dependensi eksternal.

## Fitur Utama

- Tambah tugas baru dengan validasi input (tidak boleh kosong atau hanya spasi)
- Tampilkan daftar tugas berurutan sesuai waktu penambahan
- Tandai tugas selesai / belum selesai (toggle)
- Hapus tugas individual
- Filter tugas berdasarkan status: Semua / Belum Selesai / Selesai
- Hapus semua tugas selesai sekaligus
- Persistensi data otomatis via LocalStorage

## Target Pengguna

Pengguna yang ingin mencatat dan melacak tugas harian secara sederhana langsung di browser.

## Cara Menjalankan

Tidak perlu build tool. Cukup buka `todolist-app/index.html` langsung di browser.
